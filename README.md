# Corazón Leather

This repository controls the content of the Corazón Leather :tm: Contest website through a WordPress plugin called [Git it Write](https://www.aakashweb.com/docs/git-it-write). Site pages are published from the **pages/** folder, allowing other root folders to store non-page site content.

All data belongs to the Corazón Leather :tm: Contest publisher.
